# Rails Single Page Responsive Website


## Ruby Version
ruby 2.3.0p0

## Rails Version
rails 4.2.0

## Database
MySQL


## How to Setup Application

* mkdir project
* cd project
* git clone git@github.com:PoornimaC/rails_single_page_rwd.git
* cd rails_single_page_rwd
* bundle install
* rake db:setup
